## Asteroids
##### Sovellus on yksin pelattava Asteroids-peli. Pelissä on yksi liikkuva avaruussukkula, joka ampuu asteroideja. Avaruussukkulan suuntaa ja nopeutta voi itse säädellä.
### Python-versio
##### Sovellus on tehty Python-versiolle 3.8.
